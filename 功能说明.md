# B站视频下载助手 — 功能与能力说明

> 版本：1.0.0  
> 平台：Microsoft Edge / Google Chrome（Manifest V3）  
> 定位：在 B 站普通视频页将视频保存为本地 MP4，仅供个人学习使用

---

## 1. 产品概述

**B站视频下载助手**是一款浏览器扩展。打开 B 站普通视频页后，可通过页面右下角悬浮面板选择清晰度并下载，音视频自动合并为可直接播放的 **MP4**。

- 完全免费，无内购、无激活码  
- 不收集、不上传任何用户数据  
- 界面以中文为主（`zh_CN`），商店侧提供英文扩展名/描述  

---

## 2. 核心能力一览

| 能力 | 说明 |
|------|------|
| 视频页识别 | 自动识别当前打开的 BV / av 普通视频 |
| 信息展示 | 封面、标题、UP 主、分 P、可用清晰度等 |
| 清晰度选择 | 按接口真实可用档位展示，过滤虚假高清 |
| 单集下载 | 选定清晰度后下载当前分 P，输出 MP4 |
| 分 P 队列 | 同一 BV 下多 P 时可一键队列下载全部 |
| 进度反馈 | 真实百分比 + 阶段文字（准备 / 视频 / 音频 / 合并 / 保存等） |
| 下载控制 | 支持暂停、继续（断点续传）、取消 |
| 体积预估 | 下载前展示预计大小（以接口估算为准） |
| 登录提示 | 未登录或权限不足时给出清晰度相关提示 |
| 反馈入口 | QQ `748604487`；面板/弹窗可跳转常见问题 |

## 3. 功能详解

### 3.1 适用页面

**支持**

- `https://www.bilibili.com/video/BV…`  
- `https://www.bilibili.com/video/av…`  
- 对应移动站 `m.bilibili.com/video/…`  

**不支持**

- 番剧 / 影视剧集页（`/bangumi/…` 等）  
- 需会员解锁、付费点播等需绕过权限的内容  
- 合集（`ugc_season`，右侧系列多为不同 BV）批量下载  

> **分 P ≠ 合集**  
> - **分 P**：同一 BV 的 P1 / P2 / … → **支持**队列下载  
> - **合集**：不同 BV 组成的系列列表 → **当前不支持**批量，只下你打开的这一集  

### 3.2 下载与合并

- 从 B 站播放相关接口获取流地址（与页面播放同源环境，避免 CDN 403）  
- 高清多为 DASH：分别拉取视频轨、音频轨，浏览器内合并为 MP4  
- 低清可能为整段流（durl），直接保存  
- 输出格式固定为 **MP4**（无可选「仅音频」、无自定义封装设置）  

### 3.3 清晰度

- 选项来自接口真实返回，不编造「假 1080P / 4K」  
- 更高清晰度通常需要登录或具备相应账号权限  
- 面板以胶囊按钮展示可选档位，并显示预计体积说明  

### 3.4 下载过程控制

| 操作 | 行为 |
|------|------|
| 暂停 | 中断当前网络请求，保留已下载字节 |
| 继续 | 使用 HTTP Range 断点续传 |
| 取消 | 中止任务并重置下载状态 |
| 合并中 | 进度条继续；此阶段不提供暂停（合并在本地进行） |

DASH 场景下，音视频可在后台并行拉取；界面按阶段展示（如先视频流、后音频流，再合并）。

### 3.5 分 P 队列

- 当视频有多个传统分 P（`pages`）时，显示「队列下载全部分 P」  
- 按顺序依次下载各分 P，汇总成功数量  
- 单 P 视频不显示队列按钮

## 4. 使用入口

### 4.1 页面悬浮面板（主入口，实际下载在此完成）

位置：视频页右下角浮动按钮 → 展开面板。

面板大致包含：

1. 视频识别 / 标题信息  
2. 封面与 UP 等信息卡片  
3. 分 P 列表（多 P 时）  
4. 清晰度选择  
5. 预计大小、登录相关提示  
6. 「开始下载」；多 P 时另有「队列下载」  
7. 下载中的进度区（阶段、百分比、进度条、暂停 / 取消）  
8. 底栏：常见问题、QQ 反馈  

### 4.2 工具栏弹窗（信息入口）

点击扩展图标打开 popup：

- **在视频页**：预览封面、标题、UP、可用清晰度，并可「打开下载面板」  
- **非视频页**：三步引导说明，并提供前往 B 站的入口  
- **加载失败**：错误提示 + 刷新重试  

> 弹窗只负责查看信息与唤起面板，**不会在弹窗内直接发起下载**。

## 7. 能力边界（明确不做）

以下能力**当前版本不提供**，避免与已有功能混淆：

- 番剧页下载 / VIP 解锁  
- 合集（多 BV）一键批量  
- 仅下载音频、字幕、弹幕、封面单独导出  
- 下载历史管理、自定义输出目录（跟随浏览器下载设置）  
- 设置页、账号体系、云同步

## 8. 技术能力摘要（给开发者）

| 要点 | 说明 |
|------|------|
| 架构 | MV3；`page-agent.js`（MAIN）负责 API / CDN / 合并；`content.js`（ISOLATED）负责 UI |
| 为何 MAIN | 扩展后台直连 CDN 易 403，须与页面共享 Cookie / Referer |
| 合并 | `mp4-remux`（MIT）将 DASH 音视频封装为 MP4 |
| 通信 | 页内 `postMessage`；popup ↔ content 用 `chrome.runtime` |
| 打包 | `python scripts/pack.py` → `bilibili-downloader.zip` |

更完整的架构、协议与改动指引见仓库内 **[DEVELOPER.md](./DEVELOPER.md)**。

## 9. 快速开始

1. 打开 `edge://extensions` 或 `chrome://extensions`  
2. 开启「开发者模式」→「加载解压缩的扩展」→ 选择本扩展目录  
3. 打开任意 B 站普通视频页  
4. 点击右下角悬浮按钮 → 选清晰度 → 开始下载
