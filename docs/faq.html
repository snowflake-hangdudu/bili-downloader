需要登录 B 站吗？

扩展不强制登录。但未登录时，B 站通常只提供 360P / 480P，有时 720P。登录后可能解锁 1080P 等更高清晰度（取决于视频片源与你的账号权限，大会员可解锁更高）。

若面板显示「源最高 720P」或更低，可尝试：① 登录 B 站账号；② 刷新视频页；③ 重新打开下载面板。

为什么最高只有 720P / 没有 1080P？

  • B 站 API 返回的片源决定实际上限，扩展会过滤「虚假的 1080P」选项
  • 未登录或非大会员账号，B 站可能不开放高清片源
  • 部分 UP 主上传的源本身较低清

下载失败、403、CDN 不可用怎么办？

  1. 先播放视频 5～10 秒，再点下载（让浏览器建立 CDN 连接）
  2. 尝试切换 720P 等较低清晰度
  3. 确认已登录 B 站（如需高清）
  4. 刷新页面（F5）后重试
  5. 仍失败请通过 QQ 748604487 反馈，附上视频链接与所选清晰度

合并阶段很久 / 像卡住了？

1080P 及以上为 DASH 分轨，需在本机用 JavaScript 合并音视频。文件越大，合并越久，属于正常现象。面板会显示「合并音视频」阶段。

超过约 1.5GB 的合并会被拒绝，请选择较低清晰度。下载前可查看「预计大小」。

分 P 队列下载怎么用？

多 P 视频（合集）在面板中会显示「队列下载全部 N 个分 P」。点击后按 P1 → P2 → … 顺序依次下载，每个分 P 保存为独立 MP4 文件。

  • 「开始下载」仅下载当前选中的分 P
  • 队列下载期间可暂停 / 取消当前分 P
  • 取消会停止后续分 P，已完成的文件会保留

提示文件过大？

为保护浏览器内存，合并总大小超过约 1.5GB 时会拒绝并提示选择低清。长视频 1080P 建议分段观看或选 720P 下载。

其他下载失败

  • 合并组件未加载：刷新 B 站视频页（F5），重新加载扩展
  • 页面代理超时：刷新页面；网络不稳定时可换清晰度重试
  • 无视频流：先点击播放，再下载
  • 不支持番剧页：本扩展仅支持普通 /video/BV… 详情页
